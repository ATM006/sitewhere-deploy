# sitewhere-deploy
sitewhere自动部署脚本（Ubuntu 16.04）
[关于我们]
	email：18829897162@163.com


[使用方法]
	部署：
	$ sudo bash deploy.sh
	启动：
	$ sudo bash run.sh

[组成]
	jdk + MongoDB + hiveMQ + sitewhere

	[jdk]: 			openjdk-8-jdk
	[MongoDB]: 		mongodb-org=3.6.0
	[hiveMQ]: 		hivemq-3.0.2
	[sitewhere]: 	sitewhere-server-1.7.0

[/opt]
	hivemq-3.0.2
	sitewhere
